2. Description of the Application

Security SPY (also marketed as "Smart Lens") is an Android application that silently captures photographs using the front-facing camera when user-configured applications are opened or the device lock screen is accessed. Its purpose is to help device owners identify unauthorized access to their personal devices. The Application is designed to run entirely on the User's local device. No photographs, biometric data, or personal information is transmitted to RORE Tech servers at any time.
